### Reporting Bugs
- Use the [GitHub Issue Tracker](https://github.com/akankshagupta/unimem/issues).
- Describe the bug precisely and include steps to reproduce it.
- Attach screenshots if the issue is UI-related.

### Suggesting Enhancements
- Open an issue with the tag `enhancement`.
- Describe the "why" and "how" of your proposed feature.

### Pull Requests
1. **Fork the repo** and create your branch from `main`.
2. **Install dependencies**: `npm install && cd ui && npm install`.
3. **Make your changes**: Ensure your code follows the existing style and is TypeScript-safe.
4. **Test your changes**: Run the UniMem server and verify the UI/CLI hooks work as expected.
5. **Submit the PR**: Provide a clear description of the changes and link any related issues.
